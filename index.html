Automação Residencial

Projeto desenvolvido usando Arduino e RPC para automatizar e controlar os componentes de uma residência

Sobre o projeto

O projeto realizado utiliza o conceito de comunicação RPC para controlar os aparelhos eletrônicos de uma residência. Os aparelhos eletrônicos usado nesse projeto foram componentes do Arduino, como leds, sensor de proximidade, sensor de luminosidade e outros;
